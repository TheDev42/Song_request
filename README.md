# Song_request
A simple page that allows users to request song(s) to a DJ.

## Usage - Users
Search for a song . Press ```Request```.<br>
The request is sent to the DJ.

## Usage - DJ
Enter ```./dj```.<br>
I.e. <br><br>

The page updates periodically.
